## 🤖 Introduction <a name="introduction"></a>

This is my personal 3D developer portfolio, showcasing my skills, experience, and projects through an immersive and interactive web experience. Built with React, Three.js, and Tailwind CSS, it features 3D models, animated sections, and a contact form powered by EmailJS.

---

## ⚙️ Tech Stack <a name="tech-stack"></a>

- **React.js** — UI library
- **Three.js** — 3D rendering
- **React Three Fiber** — React renderer for Three.js
- **React Three Drei** — Useful helpers for React Three Fiber
- **Framer Motion** — Animations
- **EmailJS** — Contact form email functionality
- **Vite** — Build tool
- **Tailwind CSS** — Styling

---

## 🔋 Features <a name="features"></a>

- **3D Hero Section** — Features an interactive 3D desktop model as the landing visual.
- **Experience & Work Sections** — Animated with Framer Motion for an engaging user experience.
- **3D Skills Section** — Showcases technologies using 3D geometries via Three.js and React Three Fiber.
- **Projects & Testimonials** — Animated cards using Framer Motion.
- **Contact Section with 3D Earth Model** — Interactive globe with a working email form via EmailJS.
- **Animated Star Background** — Randomly generated stars using Three.js.
- **Fully Responsive** — Optimized for all screen sizes and devices.
- **Consistent Animations** — Cohesive motion design throughout the site.

## 🤸 Quick Start <a name="quick-start"></a>

Follow these steps to run the project locally on your machine.

### Prerequisites

Make sure you have the following installed:

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/)

### Clone the Repository

```bash
git clone https://github.com/CodeSageAbhijit/abhijeet-kad.dev.git
cd abhijeet-kad.dev
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the portfolio.

---

## 🔐 Environment Variables <a name="environment-variables"></a>

Create a `.env` file in the root of the project and add the following:

```env
REACT_APP_EMAILJS_USERID=your_emailjs_user_id
REACT_APP_EMAILJS_TEMPLATEID=your_emailjs_template_id
REACT_APP_EMAILJS_RECEIVERID=your_emailjs_receiver_id
```

You can get these credentials by signing up at [EmailJS](https://www.emailjs.com/).
